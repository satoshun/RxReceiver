# RxReceiver

RxJava binding APIs for Android Broadcast Receiver.


## Usage

Please see sample project(rx-receiver-sample/).

## Install

Download via Gradle

```groovy
compile 'com.github.satoshun.rxreceiver:rx-receiver:0.1.0'
```
